>### E Commerce Website using React+tailwind
Built a responsive, user-friendly online grocery store leveraging modern front-end technologies. The platform offers intuitive product categorization, powerful search functionality, engaging promotional banners, and a fully functional shopping cart, delivering a smooth and efficient shopping experience.

### Technologies used
- React
- Tailwind CSS
